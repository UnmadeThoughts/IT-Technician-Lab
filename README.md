# IT Technician Practice Lab

## Overview
This repository documents a self-directed IT Technician practice lab focused on
hands-on troubleshooting, system configuration, and technical documentation.

## Skills Demonstrated
- Windows 10/11 installation from ISO
- BIOS/UEFI configuration
- Disk partitioning and formatting
- Driver installation and troubleshooting
- Malware detection and removal
- System imaging and recovery
- Technical documentation and logging

## Lab Environment
- Operating System: Windows 10/11
- Hardware: PC / Laptop / Virtual Machine
- Tools: Windows Defender, BIOS/UEFI, Recovery Environment

## Work In Progress
This lab is actively being built and updated as part of an accelerated IT Technician
skills refresh.
